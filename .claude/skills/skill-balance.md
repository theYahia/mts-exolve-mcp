# Проверка баланса MTS Exolve

Используй инструмент `get_balance` для проверки текущего баланса аккаунта.

## Параметры

Не требуются. Инструмент вызывается без параметров.

## Пример

```
Проверь баланс MTS Exolve
Сколько денег на счёте Exolve?
```

Ответ содержит `balance` (текущий баланс) и `credit_limit` (кредитный лимит).
Данные обновляются раз в минуту.
